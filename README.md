# Сириус.Аренда

REST API для бронирования переговорных комнат и пространств Университета «Сириус».

## О проекте

Сервис позволяет:
- Управлять пространствами (CRUD);
- Бронировать комнаты на конкретное время с проверкой доступности;
- Просматривать расписание комнат на день;
- Искать свободные комнаты.

Проект разработан в рамках хакатона "Всероссийский ИТ-раунд" (направление - программирование).

## Быстрый старт

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/arcoder0/sirius-rent.git
cd sirius-rent
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустите приложение
```bash
uvicorn main:app --reload
```

### 4. Откройте Swagger UI
Для этого перейдите по [этой ссылке](http://localhost:8000/docs).

## Структура проекта
```
database.py          # Работа с SQLite (подключение, инициализация)
```

## Выбор технологий
- **Язык программирования** - *Python 3.12*
- **Веб-фреймворк** - *FastAPI*
- **База данных** - *SQLite*
- **Хеширование паролей** - *bcrypt*
- **Валидация данных** - *pydantic*
- **Тестирование** - *FastAPI TestClient*

## Эндпоинты
| Эндпоинт | Описание |
|:---------|:---------|
| **POST** /account | Регистрация пользователя |
| **POST** /rooms | Создание комнаты |
| **GET** /rooms?capacity=...&equipment=... | Список всех комнат (с фильтрацией по вместимости и оборудованию) |
| **GET** /rooms/{id} | Информация о конкретной комнате |
| **PUT** /rooms/{id} | Обновление данных комнаты |
| **DELETE** /rooms/{id} | Удаление комнаты |
| **POST** /bookings | Создание брони |
| **DELETE** /bookings/{id} | Отмена брони |
| **GET** /rooms/{id}/bookings?date=YYYY-MM-DD | Список броней комнаты на дату |
| **GET** /rooms/available?start=...&end=...&capacity=... | Поиск свободных комнат в промежуток времени |

## Docker
Собрать и запустить проект в контейнере:
```bash
docker compose up --build
```
После запуска API будет доступно [по этому адресу](http://localhost:8000/).

## Тестирование
Для запуска тестов пропишите одну из команд:
```bash
python tests.py
```
```bash
pip install pytest
pytest tests.py -v
```

Реализовано 16 тестов, покрывающих основную логику проекта.
